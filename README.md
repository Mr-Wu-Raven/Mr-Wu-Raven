### 🎑 <i style="color:#D4B878;">寒灯栖瘦影</i>

<i style="color:#D4B878;">霜风叩破扉，孤鸿沉远渚，残月照荒陂。</i>
<i style="color:#D4B878;">冷雨侵孤馆，空山咽断笛，寒烟封古渡。</i>

<br>

<i style="color:#D4B878;">霜锁寒阶，云埋远津，烟散前盟，雨啼离魂。</i>
<i style="color:#D4B878;">灯烬更阑，酒寒愁殷，雁没沧波，笛怨孤村。</i>

<br>

<i style="color:#D4B878;">一琴一剑，踏破红尘，是非不问，浮名不存。</i>
<i style="color:#D4B878;">四海孤游，无需相闻，半生寥落，一世狂真。</i>

<br>

<i style="color:#D4B878;">一剑一酒，独走江湖。风月不问，江湖不归。</i>
<i style="color:#D4B878;">千山独行，不必相送。一身风霜，半世疏狂。</i>

<br>

<i style="color:#D4B878;">诗写千山月，酒斟一樽秋。笔底藏山河，杯中藏九州。</i>
<i style="color:#D4B878;">诗酒寄浮生，孤行任遨游。</i>

<br>

<i style="color:#D4B878;">风月从来皆薄幸，山河依旧少温情。</i>
<i style="color:#D4B878;">冷月照尽天涯路，孤身踏遍世间寒。</i>
